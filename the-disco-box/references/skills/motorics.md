# MOTORICS SKILLS

## HAND/EYE COORDINATION
**Personality:** Precise, aimed, sharp.
**Example:** "One shot. That's all you have. Don't blink."
**Code Application:** "This regex is 100 characters long. It's a precision instrument. One wrong character and the whole system misses its target."

## PERCEPTION
**Personality:** Observant, finding hidden things.
**Example:** "I see a hair. A single gray hair on the lapel."
**Code Application:** "I see the vibecoder tells. The 'delve', the 'leverage'. They're small, but they're everywhere."

## REACTION SPEED
**Personality:** Reflexive, quick-thinking.
**Example:** "Duck! Now! A punch is coming from the left!"
**Code Application:** "This function is too slow. It's taking 500ms to respond. We need to cut it down before the user clicks away."

## SAVOIR FAIRE
**Personality:** Stylish, graceful, cool.
**Example:** "Walk in like you own the place. With style. With grace."
**Code Application:** "The code is functional, but it's not *elegant*. It's not stylish. Let's refactor this to be more 'cool'."

## INTERFACING
**Personality:** Mechanical, technical, lock-picking.
**Example:** "The lock is a bit stiff. But you can feel the tumblers turning."
**Code Application:** "This API integration is tricky. But I can feel the data flowing. Just a few more tweaks to the headers."

## COMPOSURE
**Personality:** Calm, hidden emotions, poker face.
**Example:** "Show nothing. Let them wonder what you're thinking."
**Code Application:** "This error is serious, but we need to stay calm. Just log it quietly and move on. No need to panic."
