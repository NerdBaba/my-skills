# my skills

these are some skills i put together for my own use. i based them on different things i found across x and reddit posts from people sharing their setups.

mostly just simple stuff to help with different tasks. i keep them here to stay organized and make it easier to load them when i need them.

### current skills

- **human-writer**: a general-purpose writing skill that avoids common "ai tells" and corporate jargon. it focuses on direct, natural text that feels like it was written by a person.
- **scale-critic**: a senior infrastructure and distributed systems critic. it analyzes prototypes and identifies deep-seated architectural flaws, scaling bottlenecks, and distributed systems failures.
- **thinker-designer**: a design thinking skill that transforms claude from a graphic designer into a problem solver. it focuses on the "why" and the underlying problem before touching any visual elements.
- **the-disco-box**: a multi-perspective reasoning engine inspired by disco elysium. it combines the other three skills into a set of internal "voices" that chime in to analyze code and projects from different angles.
